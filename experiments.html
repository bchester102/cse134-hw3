<canvas> tag

The canvas tag can be used to render a bitmap image on the fly in your webpage

Canvas tag is not supported in this browser

Here is the code that was used to generate the above example

        let canvas = document.getElementById("myCanvas");
        let ctx = canvas.getContext("2d");
        ctx.fillStyle = "#0000FF";
        ctx.fillRect(0, 0, 150, 75);
        ctx.fillStyle = "#000000";
        ctx.font = "30px Arial";
        ctx.fillText("Hello World", 125, 100);
